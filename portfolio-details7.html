Text Evaluation

   1. Text style
   The text style used is appropriate for the message since it is in block letters with a sans serif font.

   2. Text color
   The text color is appropriate for the message since it used black over the yellow background, which are the colors used for cautionary warnings.

   3. Text size and spacing
   The text size is appropriate but the spacing isn't. The leading is off since there is too much space between the first and second lines. The kerning is also uneven because the space between letters A and L is excessive and it doesn't match the spacing of the other letters.

   4. Do you think this text follows the rules of appropriate text design? Why or why not?
   No, not all of them. The spacing is imbalanced, specifically, the appropriate leading and kerning aren't followed. Everything else observes the rules of text design, though.
